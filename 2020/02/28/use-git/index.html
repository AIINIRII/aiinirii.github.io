一、git的安装

首先，在执行git有关的命令行之前，你的电脑上需要安装一个叫git的软件。

Linux：

​ 命令行输入：

# apt-get install git

​ 或者：

# yum install git

Windows:

​ 在这个界面选择适合你电脑的版本。

​ 如果下载太慢，也可以从这里下载并安装。

MaxOS:

​ 在这个界面进行下载及安装。

安装过程就不赘述了。

二、git的使用

对于Linux和MacOS用户，打开终端便可直接使用git。

而对与Windows用户来说，如果你已经安装好了git，你需要做的就是在任意文件夹内右键，并选择Git Bash Here选项。

到目前为止，git的界面应该已经打开了，接下来的命令便是所有设备上通用的了。

三、远程仓库操作

1. 这里先介绍一些常用的有关远程仓库操作的git指令

   # 下载远程仓库的所有变动
   $ git fetch [remote]

   # 显示所有远程仓库
   $ git remote -v

   # 显示某个远程仓库的信息
   $ git remote show [remote]

   # 增加一个新的远程仓库，并命名
   $ git remote add [shortname] [url]

   # 取回远程仓库的变化，并与本地分支合并
   $ git pull [remote] [branch]

   # 上传本地指定分支到远程仓库
   $ git push [remote] [branch]

   # 强行推送当前分支到远程仓库，即使有冲突
   $ git push [remote] --force

   # 推送所有分支到远程仓库
   $ git push [remote] --all

   #简单查看远程---所有仓库
   $ git remote 

   #查看单个仓库
   $ git remote show [remote-branch-name]

   #新建远程仓库
   $ git remote add [branchname]  [url]

   #修改远程仓库
   $ git remote rename [oldname] [newname]

   #删除远程仓库
   $ git remote rm [remote-name]

   #获取远程仓库数据
   $ git fetch [remote-name] (获取仓库所有更新，但不自动合并当前分支)
   $ git pull (获取仓库所有更新，并自动合并到当前分支)

   #上传数据，如git push origin master
   $ git push [remote-name] [branch]

   #下载一个版本库
   $ git clone <版本库的网址>
   $ git clone <版本库的网址> <本地目录名>

2. 创建项目并连接仓库

  1. 首先你需要有个github或者gitlab账号。

  2. 接下来，你需要进行本地账号的配置：

    $ git config --global user.name = "你的用户名"
$ git config --global user.email = "你的邮箱"

    注意，其中用户名可以随便填写，但是邮箱一定要填对。

  3. 本地创建一个项目

    在项目根目录下执行以下操作：

    $ git init
$ echo "# 我的git项目" >> README.md
$ echo "*.log" >> .gitignore
$ git add .
$ git commit -m "initial commit"

  1. 接下来你需要在github或者gitlab上创建你自己的版本库。

  2. 添加远程主机地址：

    $ git remote add orgin <版本库的网址>

    例如：

    $ git remote add orgin https://github.com/jackfrued/Python-100-Days.git

  1. 推送项目到仓库：

    $ git push -u origin master

3. 关于多人协同开发(简易版本)

  1. 首先需要由某个人创建好以上的仓库，并将项目上传至master分支

  2. 接下来，比如我是小A，我要加入这个项目并协同开发。那么我需要做的是以下几步：

    1. 下载当前版本到本地：

      $ git init .
$ git clone <版本库的网址>

    2. 新建一个分支：

      $ git checkout -b <new_branch_name>
#用git branch命令查看现在所处分支
$ git branch

      ​ 注意：千万不要用master分支直接进行开发，一般master分支只作整合用。

    3. 开发完成后可将这个分支上传：

       $ git add .
 $ git commit -m "YOUR COMMIT"
 $ git push

  3. 最后由组长进行merge工作：

     $ git checkout master
 $ git merge <new_branch_to_merge>
 $ git commit -m
 $ git push

  4. 所有成员更新最新版本代码库：

    $ git checkout master
$ git pull
